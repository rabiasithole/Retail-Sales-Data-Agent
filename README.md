# Retail-Sales-Data-Agent
The Retail Sales Data Agent is an AI-powered assistant that simplifies retail sales analysis by allowing users to ask questions in natural language instead of writing SQL queries. 
It analyses sales transactions to provide insights into sales performance, customer behaviour, product performance, customer spending patterns, and revenue trends. 

# Project objective :
📈 Identify sales trends quickly and accurately.
🛍️ Understand customer purchasing behaviour.
📊 Compare product performance across different categories.
💡 Generate actionable business insights from sales data.
🎯 Enable informed, data-driven business decision-making.
🤖 Simplify retail data analysis through an AI-powered agent.
